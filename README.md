# RedirectProfile

在linux 和 Windows 上测试过  
already test in Linux and Windows  
  
游戏测试版本： v0.107.1

### 功能
在暂停菜单添加SL按钮

### 使用方法
#### Linux
解压压缩包, 将dll和json文件放入`~/.local/share/Steam/steamapps/common/Slay the Spire 2/mods`文件夹  
steam启动参数填`env LD_PRELOAD=/usr/lib64/libgcc_s.so.1 %command%`

#### Windows
解压压缩包, 将dll和json文件放入`<SteamInstallPath>/steamapps/common/Slay the Spire 2/mods`文件夹
